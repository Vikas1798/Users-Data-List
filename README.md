# Task Manger App

## Tech Stack
- HTML/CSS
- Tailwind,
- JavaScript (ES6)
- React + Vite
- Ant Design
- Vercel for deployment

### Installation:
1. Clone the Repository
    - https://github.com/Vikas1798/Users-Data-List.git

2. Navigate to the Cloned Repository
    - cd your-repo-name

3. Install Dependencies
    - yarn

4. Start the Project
    - yarn dev
